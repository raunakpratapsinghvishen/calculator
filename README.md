Calc-App-2025
A simple yet fully functional calculator built using HTML, CSS, and JavaScript. This project allows users to perform basic arithmetic calculations (addition, subtraction, multiplication, and division) with an easy-to-use interface.

Features
Basic Operations: Addition, subtraction, multiplication, and division.
Responsive Design: Mobile-friendly and works seamlessly on both desktop and mobile devices.
Interactive Interface: Real-time calculations with a smooth, responsive design.
Technologies Used
HTML: For building the structure of the app.
CSS: For styling the app and creating a responsive layout.
JavaScript: For implementing logic and interactive features.
